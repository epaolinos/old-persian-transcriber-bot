# Old Persian Bot

**(@OldPersianTranscriberBot)**

A specialized linguistic tool developed in Go for transcribing Latinized Old Persian into the original cuneiform script.

## 🏺 Linguistic Features

- Syllabic Logic: Correctly identifies and converts contextual signs (e.g., di, mi, du, mu) which possess unique cuneiform representations.
- Logographic Support: Scans for stems of high-frequency royal and religious terms (King, God, Country, Earth) and suggests their respective logograms.
- Numerals: A built-in converter for the additive numeric system, allowing users to see how numbers were written in Persia of the 6th century BCE.
- Simplified Input: Supports intuitive "apostrophe" notation (e.g., s' for š) for quick typing on modern mobile keyboards.
